# CV Placeholder

This folder should contain your CV file.

## Instructions

1. **Export your CV as PDF**
   - Name it: `cv.pdf`
   - Place it in the `public/` folder

2. **The download link is in:**
   - `src/components/About.tsx`
   - Look for: `<a href="/cv.pdf" download>`

3. **Recommended CV Format:**
   - PDF format (not Word doc)
   - File size: < 2MB
   - Professional academic CV layout

4. **Alternative:**
   - If you don't want a downloadable CV, remove or comment out the download button in `About.tsx`

---

**Tip:** Many academics use LaTeX for CVs. Try:
- [Overleaf CV Templates](https://www.overleaf.com/latex/templates/tagged/cv)
- [Academic CV Template](https://www.overleaf.com/latex/templates/academic-cv-template/pybsxxfhdcyx)
